# 🌉 나만의 프로필 페이지 만들기 (Littly 만들기)

## 📝 프로젝트 개요
- 이 프로젝트는 나만의 프로필 페이지인 리틀리를 만들어보는 프로젝트이다. 자기소개와 다양한 링크를 추가하여 프로필에 대해 상세하게 알려줄 수 있도록 하였다.
- 또, 나를 표현할 수 있는 명언 텍스트와 사진들, 그리고 영상을 추가하여 더 다채롭게 보일 수 있도록 하였다.
